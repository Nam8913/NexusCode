# NexusGraph — Báo Cáo Thay Đổi 27/06/2026

## Tổng Quan

Phiên bản cập nhật lớn cho **NexusGraph** — visualization tool cho code graph. Giải quyết ~15 issues từ bug fixes đến tính năng mới.

---

## Bug Fixes

### 1. Một số Edge Type không hiển thị
- **Nguyên nhân**: `EDGE_COLORS` chỉ có 11/27 edge types. `GraphController` Export chỉ có 8.
- **Fix**: Thêm đầy đủ 27 edge types vào `colors.ts` + `GraphController.cs` với màu sắc hợp lý.

### 2. Graph random khi toggle Node/Edge types
- **Nguyên nhân**: Mỗi lần toggle, `renderGraph()`kill Sigma rồi tạo lại với `Math.random()` positions.
- **Fix**: Dùng `hidden` attribute trên graphology nodes/edges thay vì recreate graph. Toggle chỉ cập nhật visibility.

### 3. Graph quá loạn, nodes chồng chéo
- **Nguyên nhân**: Không có layout algorithm — nodes đặt random.
- **Fix**: Áp dụng **ForceAtlas2** layout (async, 80 iterations qua `requestAnimationFrame`). Nodes phân tán tự nhiên.

### 4. Search xóa text → toàn bộ labels hiển thị
- **Nguyên nhân**: `handleSearch("")` set `highlighted=true` cho tất cả nodes.
- **Fix**: Query rỗng = clear `highlighted` attribute. Không render labels cho nodes không match.

### 5. Calls edges không render trong Graph
- **Nguyên nhân**: `SyntaxWalker.cs` dùng `GraphNodeId.FromFullName()` (SHA256 32 bytes) cho Calls edge source/target, nhưng nodes dùng `GetSymbolId()` (MD5 GUID 16 bytes). ID không match → edges bị skip.
- **Fix**: Calls edges giờ dùng `containingSymbol.Id.ToByteArray()` và `GetSymbolId(methodSymbol).ToByteArray()` — cùng ID scheme với nodes.

### 6. Placeholder nodes cho external types (chấm đen)
- **Nguyên nhân**: Inherits edges tới base classes external (MonoBehaviour, System.Object) không có node → placeholder nodes hiển thị truncated base64 ID.
- **Fix**: Backend thêm `sourceLabel`/`targetLabel` vào edge export. Frontend dùng label thật (tên class) thay vì ID ngẫu nhiên.

### 7. Dashboard toggle project không affect Graph/Symbols/Search
- **Nguyên nhân**: Graph filter check key `Repository`/`Project` trong metadata — nhưng nodes không có key này.
- **Fix**: Graph filter check `Object.values(meta).join(' ')` chứa tên disabled project. Symbols/Search filter theo `filePath`.

### 8. Graph không apply project filter khi load lần đầu
- **Nguyên nhân**: `initGraph()` build graph nhưng không chạy `disabledProjects` filter. `useEffect` chỉ trigger khi set thay đổi.
- **Fix**: Thêm đoạn filter `disabledProjects` trực tiếp trong `initGraph()` sau khi build graph.

### 9. Dashboard không hiện danh sách projects
- **Nguyên nhân**: `IndexController` không đăng ký repo vào `MultiRepoManagerStatic` sau index. Dashboard gọi `MultiRepoController.list()` trả rỗng.
- **Fix**: `NexusIndexService.SavePath()` ghi JSON config + thêm repo vào `MultiRepoManagerStatic.Instance`. Khi restore từ DB, đọc JSON và đăng ký lại.

### 10. DB đã có từ trước không hiện trong Dashboard
- **Nguyên nhân**: `NexusIndexService.RestoreFromDatabase()` load data nhưng không register repo vào MultiRepoManager.
- **Fix**: Thêm `indexed_repos.json` config. Khi restore → đọc paths → register repos.

### 11. Xóa project không clean JSON config
- **Nguyên nhân**: `MultiRepoController.RemoveRepository()` chỉ xóa khỏi manager, không cập nhật file.
- **Fix**: Thêm `NexusIndexService.RemoveSavedPath()` — đọc JSON, filter bỏ path, ghi lại.

### 12. Search highlight nodes đang ẩn
- **Nguyên nhân**: `handleSearch()` không check node có bị `hidden` bởi project filter không.
- **Fix**: Check `graph.getNodeAttribute(key, 'hidden')` — skip nếu true.

## Tính Năng Mới

### 1. Dashboard Project Toggle
- Danh sách projects đã index với nút **ON/OFF** bên phải
- Border trái xanh khi active, xám khi tắt
- Toggle tắt → projects loại trừ khỏi Graph, Symbols, Search
- Nút **×** xóa project (xóa khỏi manager + JSON config)

### 2. Node/Edge Type Counts
- Sidebar Graph hiện số lượng nodes/edges cho mỗi loại
- Chỉ hiện types có count > 0

### 3. Async Force Layout
- Layout chạy theo batch (10 iterations/frame) qua `requestAnimationFrame`
- Không block UI khi graph lớn
- Hiện "Layouting..." indicator khi layout đang chạy

### 4. Placeholder Nodes cho External Types
- External types (MonoBehaviour, etc.) hiện tên thật thay vì base64 ID
- Color `#484f58`, size 3, kind `External`

### 5. DB Persistence Config
- `indexed_repos.json` lưu danh sách paths đã index
- Auto-restore repos khi app khởi động
