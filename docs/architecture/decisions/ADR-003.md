# ADR-003: React + Sigma.js for Graph Visualization

## Context

Need interactive graph visualization similar to GitNexus. Previous Blazor UI had build issues with .NET 10.

## Decision

Use React + TypeScript + Sigma.js + Graphology as a separate frontend project consuming NexusCode API.

## Alternatives Considered

1. Blazor Server - Rejected: SDK/workload issues on .NET 10
2. Blazor WASM - Rejected: Same SDK issues
3. Plain HTML + JS - Rejected: Poor developer experience
4. Vue/Svelte - Considered but React chosen for ecosystem

## Consequences

- ✅ Better graph visualization (Sigma.js/WebGL)
- ✅ Separation of concerns (frontend/backend)
- ✅ Independent deployment
- ❌ Additional build tooling (Node.js, npm)
- ❌ Cross-origin requests (solved by Vite proxy)
