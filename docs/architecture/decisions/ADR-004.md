# ADR-004: SQLite for Local-First Storage

## Context

Need persistent storage for indexed data. PostgreSQL adds deployment complexity.

## Decision

Use SQLite as primary storage with InMemory fallback for graph operations.

## Alternatives Considered

1. PostgreSQL - Rejected: deployment complexity
2. Redis - Rejected: additional dependency
3. File-based JSON - Rejected: slow queries

## Consequences

- ✅ Zero configuration
- ✅ Single file database
- ✅ Works offline
- ❌ No concurrent write support
- ❌ Limited query capabilities
