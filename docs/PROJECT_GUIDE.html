NexusCode — Developer Guide

A Roslyn-based Code Intelligence Platform for C# and Unity projects. Provides Knowledge Graph, Symbol Search, MCP integration for AI agents, and Graph RAG.
Nền tảng Trí tuệ Code dựa trên Roslyn cho dự án C# và Unity. Cung cấp Knowledge Graph, Tìm kiếm Symbol, tích hợp MCP cho AI agent, và Graph RAG.

1. Project Overview

Purpose

Mục đích

Give AI agents and developers deep, semantic understanding of C# and Unity codebases — beyond text search, into symbol relationships, inheritance hierarchies, call graphs, and dependency chains.

Cung cấp cho AI agent và developer khả năng hiểu sâu, ngữ nghĩa codebase C# và Unity — vượt qua tìm kiếm text, vào mối quan hệ symbol, thứ tự kế thừa, đồ thị gọi, và chuỗi phụ thuộc.

Target Users

Người dùng mục tiêu

  • AI coding assistants connecting via MCP protocol
  • Trợ lý AI lập trình kết nối qua giao thức MCP
  • Developers using the REST API or NexusGraph web UI
  • Developer sử dụng REST API hoặc NexusGraph web UI
  • Unity teams needing project structure analysis
  • Nhóm Unity cần phân tích cấu trúc dự án

Main Features

Tính năng chính

  • Roslyn-based semantic code analysis (not regex)
  • Knowledge graph: 22 node kinds, 24 edge kinds
  • 12 MCP tools for AI agent integration
  • Fuzzy symbol search with trigram indexing
  • Graph RAG via Ollama
  • Unity analysis (MonoBehaviour, Prefabs, Scenes, Addressables)
  • Multi-repository search and comparison
  • Blast radius / impact analysis with risk scoring

4. Request Lifecycle

MCP Tool Request

Yêu cầu MCP Tool

AI Agent → JSON-RPC stdin → MCP Server
  → ResolveSymbolByName (fuzzy resolution)
  → Handler (FindReferences, FindCallers, ...)
  → SymbolSearchEngine → SymbolTable + KnowledgeGraph
  → JSON-RPC stdout → AI Agent

REST API Request

Yêu cầu REST API

Client → HTTP → Controller → NexusIndexService
  → SymbolSearchEngine / SymbolTable
  → JSON response → Client

6. Core Modules

SymbolTable

In-memory store with 7 concurrent dictionary indexes for multi-key lookup.

Bộ nhớ trong với 7 chỉ mục từ điển đồng thời để tra cứu đa khóa.

IndexLookupTra cứu
_symbolsByIdBy GUID
_symbolsByFullNameglobal::GameService
_symbolsByNameGameService
_symbolsByKindType, Method, Property...
_symbolsByFileBy file path
_symbolsByContainerMembers of a type
_referencesReferences by symbol

ResolveSymbol(name): exact full name → global:: prefix → name → case-insensitive → prefix → contains. Prefers Type > Method.

KnowledgeGraph

Directed graph with bidirectional adjacency lists. Node IDs: SHA256. Edge IDs: MD5(source+target+kind). 22 node kinds, 24 edge kinds.

Đồ thị có hướng với danh sách kề hai chiều. ID nút: SHA256. ID cạnh: MD5(source+target+kind). 22 loại nút, 24 loại cạnh.

SyntaxWalker

Roslyn AST visitor. Extracts: Symbols, References, GraphNodes, GraphEdges. Overrides VisitClassDeclaration, VisitMethodDeclaration, VisitInvocationExpression, etc.

Visitor AST Roslyn. Trích xuất: Symbols, References, GraphNodes, GraphEdges. Override VisitClassDeclaration, VisitMethodDeclaration, VisitInvocationExpression, v.v.

SymbolSearchEngine

Multi-tier search: Exact(1.0) → Name(0.9) → Prefix(0.7) → Trigram fuzzy(0.4). Graph traversal: FindCallers, FindCallees, FindDerivedTypes, FindImplementations.

Tìm kiếm đa tầng: Exact(1.0) → Name(0.9) → Prefix(0.7) → Trigram fuzzy(0.4). Duyệt đồ thị: FindCallers, FindCallees, FindDerivedTypes, FindImplementations.

CodeIndexer

FileScanner → RoslynEngine → Parallel.ForEachAsync → SyntaxWalker → SymbolTable + KnowledgeGraph

Unity Analyzers (6)

Unity Analyzers (6)

  • UnityAnalyzer — MonoBehaviour, serialized fields, lifecycle
  • UnityEventAnalyzer — UnityEvent connections
  • UnityPrefabAnalyzer — .prefab YAML parsing
  • UnitySceneAnalyzer — .unity scene parsing
  • UnityAddressablesAnalyzer — Addressable assets
  • UnityIntelligenceEngine — Orchestrator

7. Business Flow

Index a Repository

Index một Repository

User → index_repository(path)
  → FileScanner.ScanAsync(path) → list of .cs files
  → RoslynEngine.LoadProject() → Compilations
  → Parallel: SyntaxWalker.AnalyzeFile() per file
  → Merge into SymbolTable + KnowledgeGraph
  → Return: "N files, M symbols, K edges"

Find References

Tìm kiếm References

Agent → find_references("PlayerController")
  → ResolveSymbolByName() → SymbolEntity
  → SymbolTable.GetReferences(id) → List of ReferenceEntity
  → "12 refs: Line 429 in PlayerController.cs..."

Blast Radius

Bán kính vụ nổ (tác động)

Agent → blast_radius("X", depth=2)
  → FindCallers + FindCallees (recursive BFS)
  → FindDerivedTypes + FindImplementations
  → RiskScore = min(callers×5,50) + min(callees×2,20) + min(refs×1,15) + min(derived×10,15) + min(impls×5,10)
  → "Risk: 12/100 (MINIMAL)"

14. Extension Guide

Add a New MCP Tool

Thêm MCP Tool mới

  1. Add tool in tools/list response in Program.cs
  2. Add handler: string HandleNewTool(JsonElement? args)
  3. Thêm handler: string HandleNewTool(JsonElement? args)
  4. Add routing in tools/call switch
  5. Publish: dotnet publish -c Release -o publish/
  6. Publish: dotnet publish -c Release -o publish/

Add a New Graph Edge Kind

Thêm loại cạnh Graph mới

  1. Add to EdgeKind enum in Enums.cs
  2. Add creation logic in SyntaxWalker
  3. Add color in ColorConfig.cs
