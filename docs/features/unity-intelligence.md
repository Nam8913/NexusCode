# Unity Intelligence Feature

## Purpose

Unity-specific code analysis for MonoBehaviour, ScriptableObject, and Unity patterns.

## Components

- UnityAnalyzer: MonoBehaviour/ScriptableObject detection
- AssemblyDefinitionAnalyzer: .asmdef file parsing
- UnitySceneAnalyzer: Scene YAML parsing
- UnityPrefabAnalyzer: Prefab hierarchy
- UnityAddressablesAnalyzer: Addressable references
- UnityEventAnalyzer: UnityEvent detection
- UnityIntelligenceEngine: Orchestrates all analyzers
