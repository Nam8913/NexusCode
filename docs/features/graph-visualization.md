# Graph Visualization Feature

## Purpose

Interactive graph visualization using React + Sigma.js + Graphology.

## Architecture

```
NexusCode API ← REST → NexusGraph (React)
```

## Features

- Force-directed graph layout
- Node/edge type filtering
- Color coding by type
- Search and highlight
- Click node for details
- Zoom/pan navigation

## Dependencies

- React 18
- Sigma.js v2
- Graphology
- Vite
